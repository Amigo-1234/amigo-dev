Case Study: AMIGO Social Web App

A Firebase-powered social platform with real-time posts, profiles, and media uploads.

Problem

I wanted to build a Facebook-style social web app where users can create accounts, update their profiles, post content, and interact in real time—while keeping the UI clean and mobile-friendly.

Solution

I built AMIGO using Firebase for authentication and real-time data. Users can sign up, log in, edit their profile, upload images, and view a live feed that updates instantly.

Highlights

  • Mobile-first UI inspired by modern social apps
  • Authentication-protected pages and user flows
  • Real-time feed updates using Firestore

Key Features

  • Firebase Auth (signup/login/logout)
  • User profiles (name, bio, photo)
  • Image uploads with Firebase Storage
  • Real-time posts + interactions (likes/comments)
  • Responsive layout + dark mode

Tech Stack

  • HTML
  • CSS
  • JavaScript
  • Firebase Auth
  • Cloud Firestore
  • Firebase Storage
  • Firebase Hosting / Vercel
